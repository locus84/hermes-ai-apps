# AI Apps samples

Reusable examples for agents and humans. Keep samples here instead of active dashboard scan roots so installing the plugin does not automatically pollute the AI Apps gallery.

## Apps

- `apps/rpc-counter-demo/` — serverless RPC counter showing `index.html` + `manifest.json` + `server.py`.
- `apps/sample-rpc/` — alternate compact RPC counter demo.

To run a sample as a plugin-bundled active app during development:

```bash
cp -R samples/apps/rpc-counter-demo dashboard/dist/apps/rpc-counter-demo
```

Then refresh `/ai-apps`. If `plugin_api.py` changed, restart `hermes dashboard`.

To run it as a user app instead:

```bash
mkdir -p ~/.hermes/ai-apps/apps
cp -R samples/apps/rpc-counter-demo ~/.hermes/ai-apps/apps/rpc-counter-demo
```

The dashboard mirrors user-app static files to `dashboard/dist/user-apps/<app>/` at runtime for direct standalone URLs.

Do not commit runtime `data/` folders generated by sample apps.
